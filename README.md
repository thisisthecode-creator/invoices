## Invoice Generator (DE)

Ein moderner, statischer Rechnungs‑Generator mit Live‑Vorschau, Logo‑Upload, IBAN/BIC, Zahlungslinks (PayPal, Karte, SEPA) und direktem Druck als PDF (über den Browser).

### Nutzung
- `index.html` im Browser öffnen (Doppelklick reicht).
- Linkes Formular ausfüllen, Positionen hinzufügen, Steuern/Preise anpassen.
- Optional Logo hochladen.
- IBAN, BIC und Zahlungslinks eintragen.
- „Als PDF drucken“ anklicken und als PDF speichern.

### Features
- Modernes UI, dunkles Eingabepanel + helle, druckfertige Rechnung
- Live‑Vorschau für Nummer, Datum, Fälligkeit, Absender/Empfänger
- Positionen mit Menge, Einheit, Preis, Steuer %
- Rabatt (%) und Zusatzkosten (Versand/sonstige Gebühren)
- IBAN/BIC, USt‑IdNr., Kontaktangaben
- Zahlungslinks: PayPal, Karte, SEPA
- Logo‑Upload (PNG/SVG/JPG)
- Druckfreundliches A4‑Layout (Browser‑Print)

### Anpassung
- Farben/Layout: `styles.css`
- Logik/Berechnung/Bindings: `app.js`
- Struktur/Markup: `index.html`

### Hinweis
Dies ist kein steuerlicher Rat. Bitte lokale Anforderungen (z. B. GoBD, Pflichtangaben) prüfen und ggf. ergänzen.
